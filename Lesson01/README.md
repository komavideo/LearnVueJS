什么是Vue.js？
==============

Vue.js是一套开发Web页面的Javascript脚本框架，听起来难，其实是Web-Javascript脚本框架中最容易上手的框架，没有之一。

## 官方网站

* https://cn.vuejs.org/

## 竞争对手

* AngularJS1/AngularJS2
* ReactJS
* EmberJS
* Knockout
* 等等

https://cn.vuejs.org/v2/guide/comparison.html

## 技术前提
* HTML/CSS
* Javascript
* Node.js(极力推荐)

## 开发工具
* 记事本等文本编辑器
 - Visual Studio Code
 - Brackets
 - ATOM
* 浏览器（IE9,Chrome,FireFox,Safari）
